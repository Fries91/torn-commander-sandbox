Torn Commander Sandbox

A mobile-friendly multiplayer Magic: The Gathering Commander sandbox created for the Torn community.

The app is designed to support Commander games with 2–6 players, imported decks, private hands, synchronized battlefields, room codes and Commander-specific tracking.
