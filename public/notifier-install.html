Torn header userscript

Arena Commander Notifier

Adds a small crown to Torn’s header. It glows and shows a badge whenever a game has an open seat, and it includes quick buttons to host Commander, Brawl, or Custom games.

Install or update the notifier

Installation

1

Open this page inside TornPDA, or in a browser that has Tampermonkey or Violentmonkey installed.

2

Press Install or update the notifier and approve the userscript.

3

Reload Torn. A small crown appears in the header. Tap it to host a new game, view open games, or change notification settings.

Designed to stay lightweight

The notifier does not scan Torn's full page or load card images. It makes one small lobby request every 45 seconds while Torn is visible, slows to every 3 minutes when hidden, and uses a low-frequency header check instead of a full-page MutationObserver.

If the server cannot be reached, checks back off automatically to reduce repeated work. The popup is only rendered while it is open.

Automatic updates

The script includes a version number plus @updateURL and @downloadURL. Compatible userscript managers check the hosted file and replace older versions automatically. The notifier also checks its version and shows an update button when a newer release is available.

Keep the file address unchanged when uploading future versions. Increase the @version number each time.

Host or join from Torn

Tap the crown and choose Commander, Brawl, or Custom to open the matching host form. After creating the room, turn on List this room in the Torn notifier. Only waiting rooms with open seats are shown.
